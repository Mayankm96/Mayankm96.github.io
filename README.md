# Personal Website

Made using [end2end](https://github.com/nandomoreirame/end2end) based on Clean Jekyll theme.

## Usage:

* Fork the repository
* Rename to `username.github.io`
* Modify the `_config` file to add in your details
* Replace all the posts and images with your data
